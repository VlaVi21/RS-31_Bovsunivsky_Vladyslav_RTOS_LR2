# RS-31_Bovsunivsky_Vladyslav_RTOS_LR2

**Завдання до лабораторної роботи**: 

Варіант 2: Запрограмувати світлодіоди PD14 та PD15 з частотою повторення імпульсів 2 Гц,
тривалість імпульсу 100 мс. Зчитати стан кнопки PA0 та відобразити його на світлодіоді PD14.

Цей проєкт реалізує базову роботу з GPIO на мікроконтролері STM32 з використанням HAL-бібліотеки.
Програма керує світлодіодами та обробляє натискання кнопки.

### Функціонал:
**LED2 Pulse Task**: Генерує імпульсний сигнал на світлодіоді (PA1) з частотою 2 Гц.

**Button Control**: Опитує кнопку на PA0 (System Button).

### Підключення:
LED2 (PA1): GND → LED → резистор (~325Ω) → PA1
LED1 (PA2): GND → LED → резистор (~325Ω) → PA2
Button (PA0): GND → кнопка → PA0 (з внутрішньою підтяжкою Pull-Up)

### Як запустити:
1. Відкрийте файл `.ioc` у STM32CubeIDE.
2. Згенеруйте код (якщо потрібно).
3. Скомпілюйте та прошийте через ST-Link.
